# Sales Data ETL Pipeline

## Overview
This project implements a simple end-to-end ETL pipeline for sales data.

## Tech Stack
- Python
- SQL (PostgreSQL/MySQL)
- CSV
- Git

## Pipeline Steps
1. Raw sales data is read from CSV
2. Invalid records are filtered using Python
3. Cleaned data is written to a new CSV
4. Data is loaded into a relational database
5. SQL queries generate analytical insights

## Key Learnings
- Data cleaning and validation
- Error handling in ETL pipelines
- Writing analytical SQL queries
- Version control using Git
